## About

Extended CUA bindings for Emacs. Feels like VSCodium.

## Install

Install Doom Emacs first, per it's [instructions](https://doomemacs.org/).

Backup your doom folder and clone the repository into it. Run doom sync.

```bash
mv ~/.config/doom ~/.config/doom.bak
git clone --depth 1 git@github.com:clyfe/dot-doom2.git ~/.config/doom
~/.config/emacs/bin/doom sync
```
Start Emacs and enjoy.
